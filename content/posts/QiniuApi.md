## API参考

### Java SDK

#### 添加依赖
- Maven
``` xml
<dependency>
<groupId>com.qiniu</groupId>
<artifactId>qiniu-java-sdk</artifactId>
<version>[7.0.0, 7.1.99]</version>
</dependency>
```
- Gradle
``` groovy
compile 'com.qiniu:qiniu-java-sdk:7.x.+'
```
